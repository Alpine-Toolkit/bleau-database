    Rapporter des erreurs ou un dysfonctionnement

    • Veuillez vérifier avant que le problème n'a pas déjà été rapporté par quelqu'un d'autre. Les tickets sont répertoriés à cette adresse .
    • S'il s'agit d'un problème d'affichage, veuillez indiquer la résolution de votre écran, le navigateur utilisé ainsi que sa version.
      Notez que ce site requiert un navigateur compatible HTML5, les navigateurs ancestraux ne sont pas supporté.
      Si vous le pouvez, installez un navigateur récent (e.g. Firefox, Chrome, Safari, Microsoft Edge).
      Si vous naviguez sur un smartphone, assurez vous d'avoir une bonne connexion, un modèle suffisament performant et un navigateur compatible.
    • Pour créer un ticket, vous devez avoir un compte Github . Cela permet d'identifier les personnes et de communiquer.

    Contribuer

    Vous pouvez proposer des « pull requests » sur le dépôt . Soit en utilisant directement Git, si vous maîtriser cet outil informatique (pour en savoir plus vous pouvez consulter ce lien).

    En revanche, l'interface de Github permet d'éditer en ligne le code source. Par exemple, pour proposer une correction sur le site, vous pouvez suivre ce processus : {# (à condition d'avoir un compte) #}

    1. Connectez vous sur Github ou créez un compte.
    2. Naviguez dans le code source à partir de la page d'accueil, cliquez sur « share/WebApplication » puis sur « template », ensuite cliquez sur le fichier qui correspond à la page qui vous intéresse. Le contenu du fichier doit à présent s'afficher. Par exemple pour cette page.
    3. Ensuite pour éditer le fichier, cliquez sur l'icône représentant un crayon à droite du bouton « History ».
      4. {# https://github.com/Cosiroc/bleau-database/edit/master/share/WebApplication/templates/contribute.html #}
    4. Une fois que vous avez terminé les modifications, renseignez en dessous de « Commit changes » dans les champs une courte et éventuellement une longue description de vos modifications. Puis cochez « Create a new branch for this commit and start a pull request. » et cliquez « Commit changes ». Une notification est envoyée. Votre requête doit apparaître sur cette page.
