Données compatibles avec un Smartphone ou un GPS

Les coordonnées des massifs et des circuits sont disponible au format GPX sous forme d'une liste de points d'intérêt dans ce fichier bleau.gpx

Ce fichier peut être utilisé conjointement avec un smartphone et l'application OsmAnd qui exploite le fond de carte OpenStreetMap.
Il faut pour cela copier ce fichier sur votre smartphone dans le dossier osmand sous le nom favourites.gpx.

Pensez à sauvegarder et fusionner vos points d'intérêt avant d'écraser le fichier !

Base de Données au format JSON

La base de données de référence est contenu dans ce ficher bleau.json

Le module Python BleauDataBase offre une API orienté objet pour exploiter la base de données. Voir la page du dépôt Git pour plus de détails.

Base de Données au format GeoJSON

Les données sont aussi disponible au format GeoJSON :

  • bleau-massifs-geo.json
  • bleau-circuits-geo.json
  • bleau-places-geo.json
