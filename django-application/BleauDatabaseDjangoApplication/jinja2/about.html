    Contribuer à la base de données

    L'accès à cette application n'est pas publique et requiert l'activation d'un compte sur demande et justifié.

    Les base de données

    Il y a deux types de base de données, la base de données stockée dans le moteur de base de données PostgreSQL/PostGis accessible à partir de cette application et la base de données exportée à intervalle régulier, au format JSON par exemple.

    Interfaces d'administration

    Cette application possèdent plusieurs interfaces d'administration :

    • une interface web optimisée pour l'édition de la base de données,
    • l'interface d'administration de Django qui donne un accès de plus bas niveau,
    • l'API REST qui permet d'interconnecter la base de données (PostgreSQL) avec une autre application,
    • et en dernier recourt les pull requests au niveau du dépôt.
